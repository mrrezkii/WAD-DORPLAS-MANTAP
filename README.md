Buka CMD atau Terminal kemudian lakukan instalasi package/library/plugin dibawah dengan command sebagai berikut :

`create database dorplas`

`composer install`

`cp .env.example .env`

`php artisan key:generate`

`php artisan migrate:fresh --seed`

`php artisan serve`
